Файл миграции — текстовый файл с набором команд для внесения изменений в структуру модели. Например, для удаления или добавления полей.
Файлы миграций создаются автоматически в папке migrations каждого приложения. Каждый новый файл миграции не создает модель заново, а лишь описывает новые текущие изменения.
Всегда можно откатиться назад к любому прошлому файлу миграции

`python manage.py makemigrations` - для создания миграции нужно в терминале прописать команду 
`python manage.py sqlmigrations <appName> <MigrationNumber>` - чтобы просмотреть, какой sql запрос будет выполнен при запуске файла миграции
`python manage.py migrate` - запустить файл миграции
