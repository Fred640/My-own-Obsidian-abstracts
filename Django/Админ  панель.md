Настройка админ панели
в `settings.py` есть переменная `language_code` которая отвечает за язык админ панели



##### Добавление пользователей:

после добавления первого sudo пользователя остальных пользователей можно добавлять и распределять по группам через созданного sudo пользователя в админ панели
###### sudo пользователь:
1) `python manage.py createsuperuser`
2) ввести имя
3) ввести почту
4) ввести пароль и повторить
##### Использование админ панели
регистрирование моделей 
```
@admin.register(<ModelName>)
class <ModelName>Admin(admin.ModelAdmin):
	list_display = ('id', "name", 'isPublished')     #поля которые буду отображаться на странице
	list_display_links = ["name"]                    #поля которые будут ссылками на страницу записи
	ordering = ['id']                                #поле по которому будут сортироваться записи на странице модели
	list_editable = []                               #список полей которые можно редоктировать 
	list_per_page = 4                                #количетво отображаемых записей на одной странице
```
