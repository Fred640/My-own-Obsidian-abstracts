Родительские шаблоны должны находиться в папке `<projectName>/templates` чтобы их в дальнейшем использовать нужно в `settings.py` в коллекции `templates` добавить
`'DIRS': [BASE_DIR / '<ProjectName>/templates',]`

Для того чтобы создать родительский шаблон, нужно в обычный шаблон добавить теги block. Ниже родительский шаблон определяет три блока которые в дальнейшем могут быть заполнены дочерними шаблонами

```
#родительский шаблон
<!DOCTYPE html>
<html lang="en">
	<head>
	<meta charset="utf-8" />
	<meta name="viewport" content="width=device-width, initial-scale=1" />
	<title>
		{% block title%}default_title{% endblock title%}
	</title>
	</head>
	<body>
		<div id="header">
			{% block header_content %}
				<nav>
					<ul>
					<li>1</li>
					<li>2</li>
					<li>3</li>
					<ul>
				</nav>
			{% endblock %}
			</div>
		<div id="body">
			{% block body_content %}
				default_body
			{% endblock %}
		</div>
	</body>
</html>
```

```
#дочерний шаблон

{% extends "base.html" %}
{% block title %}children_title{% endblock %}
{% block header_content %}asd{% endblock %}
{% block body_content %}{% lorem 6 w random %}{% endblock %}
```
если в дочернем шаблоне не будет переопределено значение родительского block, то будет использоваться контент родительского block

Переменная {{ block.super }} позволяет обратиться к содержимому блока родительского шаблона, это может быть полезно когда в дочернем шаблоне нужно только добавить содержимое к родительскому блоку, а не полностью переписать содержимое родительского блока.
```
{% extends "man/base.html" %}
{% block title %}children_title{% endblock %}
{% block body_content %} {{ block.super }} asdasdasd{% endblock %}
                        ^^^^^^^^^^^^^^^^^^^
```
