

Для того чтобы подключить статические файлы, нужно в `settings.py` указать:
 1) STATICK_URL - префикс url-адреса для статических файлов
 2) STATIC_ROOT - путь к общей статической папке которая будет сгенерирована при выполнении команды `python manage.py collectstatic` (эта команда нужна, для того чтобы собрать все статичные файлы и переместить их в одну общую папку, для того чтобы перенести сайт на постоянный сервер)
 3) STATICFILES_DIRS - путь ко всем нестандартным папкам со статичными файлами
